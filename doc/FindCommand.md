# FindCommand

## Main

- Find password which you want

### Help

```
Usage:
  find [options] [--] [<keyword>]

Arguments:
  keyword               Get password by the password keyword.

Options:
  -a, --list            Show passwords list.
```
#### Params info

```
keyword     Get password by the password keyword.
-a|--list   Show passwords list.
```
