# FindCommand

## 主要功能

- 查找密码项

### Help

```
Usage:
  find [options]

Options:
  -N, --name[=NAME]     Get password from the password name.
  -a, --list            Get all passwords of user.
```
#### 参数详解

```
-N 要查找密码项的名称
-a 列出所有的密码项名称
```
